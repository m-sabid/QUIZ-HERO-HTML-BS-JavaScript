Differences between javascript map, filter, forEach and find:

map: The map method creates a new array by applying a function to each element of the original array. It returns a new array of the same length as the original array with the result of applying the function to each element. It does not modify the original array.


filter: The filter method creates a new array with all elements that pass a test implemented by the provided function. It returns a new array containing only the elements for which the function returns true. It does not modify the original array.


forEach: The forEach method iterates over an array and executes a function for each element. It does not create a new array and does not modify the original array.


find: The find method returns the value of the first element in an array that satisfies a provided testing function. It returns undefined if no values satisfy the function. It does not modify the original array.

Differences between localStorage, sessionStorage, and cookies:

LocalStorage: localStorage is a type of web storage that allows you to store key/value pairs in the browser with no expiration date. The data stored in localStorage persists even after the browser window is closed and reopened. This makes it a good option for storing data that needs to be accessed by the user even after they close and reopen the browser.

SessionStorage: sessionStorage is similar to localStorage in that it allows you to store key/value pairs in the browser. However, data stored in sessionStorage only lasts for the duration of the user's session, which means it is lost when the user closes the browser window or tab. This makes sessionStorage a good option for storing temporary data that does not need to persist after the user leaves the website.

Cookies: cookies are small text files that are stored on the user's computer by the browser. Cookies can be used to store data such as user preferences or login information. Unlike localStorage and sessionStorage, cookies can have an expiration date, which means they will be deleted after a specified amount of time. Cookies can also be accessed by both the client-side and server-side code, which makes them useful for maintaining state across multiple requests. However, cookies have a size limit of 4KB and are less secure than localStorage and sessionStorage because they can be intercepted or manipulated by third parties.

About The quizTimer function and chooseQuiz function

quizTimer: The quizTimer function is a JavaScript function that creates a timer to be used in a quiz. It takes a boolean argument called dismiss, which when set to true, clears the timer and stops the countdown.

chooseQuiz: The chooseQuiz function is a JavaScript function that allows the user to select or choose a quiz and record their answer. It takes two arguments, the index of the selected quiz and the user's answer to that quiz.
